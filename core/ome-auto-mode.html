Oh My Emacs Auto-mode

This is part of the oh-my-emacs.

This file contains various settings for minor file types, such as SCons build file, archlinux's PKGBUILD file, etc.

SCons

SCons is a next generation build system that is much easier to configure and use than the traditional make utility. SCons uses the Python general purpose programming language as a foundation, and all SCons build files are Python scripts, so we just set SCons build file work with python-mode. 

(add-to-list 'auto-mode-alist '("SConstruct" . python-mode))

Puppet

Puppet is configuration management tool from Puppet Labs, it is the de facto industry standard for IT automation. Puppet is written by ruby but has its own DSL, so the emacs builtin ruby-mode is not proper for editing puppet files. 

(defun ome-puppet-mode-setup ()
  (when (featurep 'evil)
    (add-hook 'puppet-mode-hook 'turn-on-evil-mode)))

(ome-install 'puppet-mode)

ArchLinux PKGBUILD

A PKGBUILD is an Arch Linux package build description file, which is actually a shell script. Thus you can edit PKGBUILD file with emacs shell-script-mode, however, pkgbuild-mode provides a better workflow based on shell-script-mode for PKGBUILD files, including:

  • A default PKGBUILD template as a starting point
  • Some shortcuts to update md5sum, browse package's website, build tarball, etc.

Just install pkgbuild-mode from el-get will bring all this goodies to your fingertips. 

(when (executable-find "pacman")
  (ome-install 'pkgbuild-mode))

Wikipedia

Wikitext language, or wiki markup, is a lightweight markup language used to write pages in wiki websites, such as Wikipedia, and is a simplified alternative/intermediate to HTML. Its ultimate purpose is to be converted by wiki software into HTML, which in turn is served to web browsers.

Unfortunately, there is no commonly accepted standard wikitext language. So I just pick wikipedia-mode, which is one of the many possible packages from el-get, as the mode for editing wiki documents. If you have better choice, please tell me. 

(defun ome-wikipedia-mode-setup ()
  (add-to-list 'auto-mode-alist
               '("\\.wiki\\'" . wikipedia-mode)))

(ome-install 'wikipedia-mode)

YAML

YAML is a human friendly data serialization standard for all programming languages. I really like it for its simplicity and readability. However, Emacs didn't have a builtin mode for editing YAML files, so you know how easy to solve this by adopting el-get. Thanks el-get again! 

(defun ome-yaml-mode-setup ()
  (add-hook 'yaml-mode-hook
            (lambda ()
              (linum-mode 1))))

(ome-install 'yaml-mode)
